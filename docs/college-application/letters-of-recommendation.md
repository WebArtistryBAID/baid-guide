# 推荐信

几乎所有美国大学都会要求申请者提交**推荐信 (letters of recommendation)**。其他国家的大学对推荐信的要求不一，且与美国大学的要求有较大差异。本文主要介绍美国大学推荐信的相关内容。

## 类型

* **学校简介 (school profile)。** 虽然学校简介不是推荐信的一种，但它通常和推荐信一起提交。学校简介一般由升学指导老师 (counselor) 递交，介绍学校的整体情况、课程设置、学生构成、当年和往年学生最高与最低 GPA、往年录取数据等。

* **升学指导推荐信 (counselor letter of recommendation)。** 升学指导推荐信一般由升学指导老师 (counselor) 撰写，介绍学生的整体情况、学术表现、课外活动、个人品质等。升学指导推荐信的目的是让招生官了解学生**在学校的整体表现和位置**。

* **教师推荐信 (teacher letters of recommendation)。** 教师推荐信一般由学生的老师撰写，介绍学生在课堂上的表现、学术能力、学习态度、与同学的互动等。老师还会就学生的学术成就、学术潜力、写作能力、创意思维、领导力、成熟度、尊重度、包容心等作出评分。教师推荐信的目的是让招生官了解学生**在具体学科课堂上的表现和能力**。一般
 
  * 邀请十一年级老师撰写
  * 一封来自理科 (数学和科学) 老师，一封来自文科 (语言和历史) 老师
  * 老师一般需要来自核心学科 (core subjects)，即英语、外语、数学、科学和历史

* **其他推荐信。** 有些学校允许提交其他类型的推荐信，包括艺术老师、家庭成员、同学、雇主、教授等撰写的推荐信。

许多学校要求必须提交**一封升学指导推荐信** (counselor letter of recommendation) 和 **两封教师推荐信** (teacher letters of recommendation)。一般不建议提交额外的推荐信。

## 怎么做

:::tip

新生暂时不需要考虑推荐信的事情。如果你已经进入十一年级，有必要开始做这些事情:

* **选定推荐信的老师。** 选定好后，有必要和老师熟络起来，争取让老师对你有更深入的了解。
* **记录课堂表现。** 课堂上的趣事、表现、成就等，都可以记录下来，方便以后写推荐信时使用。

十一年级末，就可以开始正式邀请老师写推荐信了。邀请时，别忘记筛选提前记录好的课堂表现，发给老师，方便老师的工作。一般而言，不同的推荐老师最好涉及不同的方面。

:::
